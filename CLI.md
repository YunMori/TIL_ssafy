# CLI란 무엇인가?
명령어를 통해서 사용자와 컴퓨터가 상호 작용 하는 방식 like a terminal

# 왜 CLI를 사용해야 할까?
1. 효율성
  - 키보드만으로 모든 작업을 빠르고 가볍게 수행할 수 있음
2. 정밀한 제어
  - 그래픽 화면에 보이지 않는 숨겨진 설정까지 정밀하게 제어할 수 있음
3. 강력한 자동화
  - 단순하고 반복적인 작업을 명령어 한줄로 자동화 할 수 있음
4. 표준 환경
  - 서버, 클라우드 등 그래픽이 없는 개발환경의 필수 도구

. = 현재 디랙토리
.. = 현재의 상위 디렉토리(부모 폴더)

절대 경로 = 긴거
상대 경로 = 짧은거
